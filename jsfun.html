The Fundamentals of JavaScript

Feb 8 2025

Recognizing the importance of understanding Javascript for browser exploitation, I've started with the basics. One has to learn to crawl before they can walk. I'm beginning my journey by getting a bit of context about JavaScript and when, where, and what it's used for.

What is JavaScript?

JavaScript is a scripting/programming language that can be used for the manipulation of the behavior of web pages. It's often used in conjunction with HTML and CSS to build web pages. Using JavaScript, one can implement complex and dynamic features (updates to content, animated graphics, etc.) so that web pages aren't as static. JavaScript is used primarily in the browser, and enables developers to manipulate webpage content through the DOM, retrieve contnet from servers using the fetch() API, store complex data, draw graphics, interact with the device running the browser through various APIs, etc. JavaScript can be used for both client-side scripts on webpages and also server-side applications.
